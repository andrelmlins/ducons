# Ducons

![NPM](https://img.shields.io/npm/l/ducons)

Biblioteca de icones Brasileira.

## **Como instalar**:

```
npm i ducons

# or

yarn add ducons
```

## **Como usar**:

```js

import Ducons from 'ducons'

export default function Componete(){

    return(
        <div>
            <Ducons name="bookmark" size={24} />
        </div>
    )
}
```
